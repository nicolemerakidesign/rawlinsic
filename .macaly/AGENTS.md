# Project: Rawlins Contact Page

## Project Context
- Building a contact page for Rawlins (rawlinsic.com) — a global consultancy at the intersection of strategy, operations, and technology.
- Design system matches the existing site at dev.rawlinsic.com/v2/index.html
- Dark navy background (#060c16), gold accents (#c9a84c, #d4b878, #e8d5a0)
- Fonts: Cormorant Garamond (headings/serif, italic accents) + DM Sans (body/UI)
- Ambient orbs, micro-particles, custom gold cursor, scroll reveal animations
